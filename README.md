# PriceCheck Internal Pricing Control Panel

Monorepo MVP blueprint:
- `backend/`: Laravel API (clean architecture, Sanctum-ready, WooCommerce integration service).
- `frontend/`: Next.js App Router admin UI with Tailwind/shadcn-style component structure.

## Modules
Auth, Dashboard, Families, Products, Review Queue, Pricing Drafts, Pricing Application, Woo Sync, History/Logs, Settings, User Management.
